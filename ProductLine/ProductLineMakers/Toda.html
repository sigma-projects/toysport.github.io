TODA RACING 4AG Competition Engine Kit

The Toda Racing 4AG competition engine kit (for use with MAP sensor equipped engines) includes a brand new Toda Racing ECU, high compression forged piston kit, cam pulleys, valve springs and camshafts.  The pistons are available in 3 sizes; 81.00mm / 81.25mm / 82.00mm.  Camshaft are one 272 (7.9mm) camshaft and one 264 (7.9mm) camshaft.          Kit Price: $3,310  

An optional remote ignition controller is available for an additional $150.  

 

TODA RACING 4AG Competition ECU

Toda Racing offers a competition fuel computer for the 4AG engine for use with MAP sensor equipped engines.  This ECU revises fuel and ignition curves and is recommended for use with Toda camshafts.          ECU Price: $1,450

 

TODA RACING 4AG Forged Piston Kit

The Toda Racing forged piston kit for the 4AG raises the compression ration to 11.0 : 1.  Pistons are available in the following sizes: 81.0 / 81.25 / 82.0       Piston Set  $880
